# CareerTuner Second Brain Viewer

이 폴더는 CareerTuner의 Obsidian wiki와 Graphify code graph를 포트폴리오용으로 보여주는 GitHub Pages 뷰어다.

- 원본 Obsidian vault와 raw Graphify 산출물은 비공개 submodule과 임시 작업 경로에 유지한다.
- 공개본은 실제 code-only Graphify 추출 수치와 repo-relative 코드 경로를 함께 보여준다.
- Wiki cluster는 118개 전체 graph node와 **31개 curated Wiki 카드**를 포함한다. 모든 graph node의 상세 panel에서 같은 ID의 Public Wiki evidence page를 직접 열 수 있다.
- 시연 준비 cluster는 admin 29-code 권한, Firebase/native identity, DB lifecycle, AWS-first outage demo, A~F evidence와 Android/iOS/Qt live gate를 연결한다.
- Provenance는 synthesis vault merge `114b6d91`와 latest projection merge `248e082b`를 분리해 표시한다.
- 전체 원본 `graph.json`은 절대경로와 대용량 edge 정보를 포함하므로 그대로 공개하지 않고, 리뷰어가 이해하기 쉬운 curated graph로 압축한다.
- API key, DB 계정값, 실제 비밀번호, 개인 credential, 로컬 절대경로는 포함하지 않는다.
