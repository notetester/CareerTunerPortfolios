# CareerTuner Public Knowledge Map

이 폴더는 공개 GitHub Pages 데모에서 제공하는 CareerTuner 포트폴리오 지식맵이다.

- 원본 Obsidian vault는 비공개 저장소에 유지한다.
- 공개본에는 API key, DB 계정값, 실제 비밀번호, 개인 credential, 로컬 절대경로를 포함하지 않는다.
- 제품 구조, 실제 코드 경로, 기술 축, AI/ML 범위, 관리자 기능, 배포 산출물은 포트폴리오 증거로 적극 공개한다.
- `SecondBrain/`은 Obsidian + Graphify + LLM Wiki 기반 agent memory, 118개 project/code node와 graph→Wiki evidence 이동을 제공한다.
- `Wiki/`는 CareerTuner 제품·기능·API·DB·AI/ML·배포·팀 구조 35개 curated article과 118개 graph evidence, 총 153개 문서를 검색, provenance, 본문·목차·관련 문서 단위로 읽는 정적 Wiki다.
