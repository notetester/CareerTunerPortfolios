# CareerTuner Feature Workspaces

This directory is the ownership map for team assignment.

Each feature owns four work areas:

- `frontend`: user-facing React screens, feature components, hooks, and API clients
- `backend`: user-facing Spring Boot APIs, services, MyBatis mappers, DTOs, and domain objects
- `admin-frontend`: administrator React screens and controls for the same feature
- `admin-backend`: administrator Spring Boot APIs, services, MyBatis mappers, DTOs, and audit controls

Do not remove feature folders without a team decision.

Runtime source paths are documented in `docs/FEATURE_OWNERSHIP.md`.
