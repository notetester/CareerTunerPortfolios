# C 모델 스크립트 운영 문서

이 폴더의 실행 코드를 보조하는 짧은 데이터·게이트 runbook과 시점별 결과 요약을 둔다. 장문 실험 보고서는 `docs/ai-reports/`, raw output은 `docs/ai-artifacts/` 서브모듈이 정본이다.

| 문서 | 용도 |
| --- | --- |
| [GATE_DATA_RUNBOOK.md](GATE_DATA_RUNBOOK.md) | gate 평가 입력·실행 절차 |
| [GATE_ADVERSARIAL_LIVE_RESULT.md](GATE_ADVERSARIAL_LIVE_RESULT.md) | 적대적 live 실행의 호환 요약 |

실행 전 상위 [CURRENT_STATE.md](../CURRENT_STATE.md)와 각 script의 `--help`를 확인한다. 문서에 과거 모델 tag가 있더라도 현재 기본값으로 간주하지 않는다.
