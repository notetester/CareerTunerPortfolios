# C 모델 판정 템플릿

오프라인 평가자가 동일한 rubric과 출력 계약으로 모델 결과를 판정하도록 만드는 prompt template을 둔다. 런타임 prompt가 아니며 사용자 요청 처리 경로에 직접 로드하지 않는다.

- [rag_hardcase_top_llm_judge_prompt_v1.md](rag_hardcase_top_llm_judge_prompt_v1.md): RAG hard-case 독립 판정용 v1 템플릿

템플릿을 바꾸면 버전을 새 파일로 올리고, 적용한 evaluation artifact와 해석 보고서에서 정확한 파일명·commit을 기록한다.
