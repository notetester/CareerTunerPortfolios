# 환경 구성 인덱스

[`environments.json`](environments.json)은 CareerTuner 환경 프로파일을 사람이 비교하기 위한 문서용 인덱스다.
애플리케이션이 이 파일을 직접 읽지는 않으며, 실제 동작은 환경변수·Compose·Spring 프로파일·Vite mode 설정이
결정한다. 우선순위와 전환 절차는 [환경 프로파일 문서](../docs/ENVIRONMENTS.md)가 정본이다.

## 관리 원칙

- 호스트나 프로파일을 바꾸면 해당 런타임 설정과 이 인덱스를 같은 변경에서 맞춘다.
- 주소는 비밀값이 아니더라도 내부 네트워크 구조를 드러낼 수 있다. 공개 포트폴리오로 내보낼 때는 내부 IP와
  장비 식별자를 별도 공개용 예시값으로 치환한다.
- API 키, 비밀번호, 토큰, OAuth secret은 이 폴더에 넣지 않는다.
- JSON 값은 서비스 가용성 보장이 아니라 구성 스냅샷이다. 운영 상태는 health/readiness와 배포 기록으로 확인한다.
